1xx: Informational - (100 for "Continue")
2xx: Success - (200 for "OK")
3xx: Redirection - (301 for "Moved Permanently")

XSS stands for Cross-Site Scripting.It's a security vulnerability where an attacker injects malicious scripts into a web application, and those scripts are executed by other users' browsers. 
A common way a site vulnerable to an XSS attack, is if it doesn't properly validate user inputs.
